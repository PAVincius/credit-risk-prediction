# Prerequisites

- An AWS account
- Basic knowledge of Python and machine learning
- Familiarity with AWS services like S3, Redshift, Athena, SageMaker, QuickSight, and CloudWatch

## A good way to get that knowledge

- [Amazon S3](https://www.coursera.org/projects/aws-s3-basics)
- [Amazon Redshift](https://www.coursera.org/learn/introduction-to-designing-data-lakes-in-aws)
- [Amazon Athena](https://www.coursera.org/learn/getting-started-data-analytics-aws)
- [Amazon SageMaker](https://www.coursera.org/projects/object-detection-sagemaker)
- [Amazon QuickSight](https://www.coursera.org/learn/getting-started-data-analytics-aws)
- [Amazon CloudWatch](https://www.coursera.org/learn/aws-amazon-cloudwatch-getting-started)
